### Problem
The game master is kidnapped! Since he is the only one who can give you the final grade, then
your grade is in risk as well!
The game master is locked in a room which he can send out an image of six-digit number code.
This number can be used to unlock the door. You all have the task to decode this number. The
number is written in E-13B font which is a font where stylised to maximise the capacity of machine
readability which maintain the readability for humans. The font set consists of characters on a 9 x 7 grid.
## Tasks
Create a program which
1. Receive an image as the input
2. Locate the character and decode it
3. Print the decoded number on the screen
Submit the program in Java, Python, or C++ (NO EXTERNAL HELPER LIBRARIES ALLOWED)
